Create a type adapter that converts generated API types to existing domain models.

This adapter will:
1. Convert generated Request types to existing request formats
2. Convert generated Response types to existing domain models
3. Handle null safety and type mismatches

Example structure:
```dart
import '../generated/openapi.dart';
import '../features/review_center/data/models/pending_transaction_model.dart';

/// Adapter for converting generated types to domain models
class TypeAdapter {
  TypeAdapter._();

  /// Convert generated transaction to domain model
  static PendingTransactionModel toPendingTransaction(TransactionDto dto) {
    return PendingTransactionModel(
      id: dto.id,
      date: dto.date,
      payee: dto.payee,
      narration: dto.narration,
      amount: dto.postings?.first?.amount?.number,
      currency: dto.postings?.first?.amount?.currency,
      status: dto.status ?? 'pending',
    );
  }

  /// Convert domain model to create request
  static CreateTransactionRequest toCreateRequest(PendingTransactionModel model) {
    return CreateTransactionRequest(
      date: model.date,
      payee: model.payee,
      narration: model.narration,
      postings: [
        PostingDto(
          account: model.account,
          amount: AmountDto(number: model.amount, currency: model.currency),
        ),
      ],
    );
  }
}
```

Note: The exact field mappings depend on the generated types structure. Adjust based on actual generated code.

Pattern to follow:
```dart
import '../../../api/api_client.dart';
import '../../../api/type_adapter.dart';
import '../../../api/generated/openapi.dart';

class ReviewCenterRemoteDatasource {
  final TransactionApi _transactionApi;

  ReviewCenterRemoteDatasource()
      : _transactionApi = TransactionApi(ApiClientWrapper.instance.dio);

  Future<List<PendingTransactionModel>> getPendingTransactions(String region) async {
    try {
      final response = await _transactionApi.listTransactions(region: region);
      return response.data
          .map((dto) => TypeAdapter.toPendingTransaction(dto))
          .toList();
    } on DioError catch (e) {
      throw _handleDioError(e);
    }
  }

  Exception _handleDioError(DioError e) {
    if (e.response?.statusCode == 401) {
      return const ApiException(401, 'Login expired');
    }
    return ApiException(e.response?.statusCode ?? 0, e.message);
  }
}
```

Important: Keep method signatures unchanged to maintain backward compatibility with repository layer.
