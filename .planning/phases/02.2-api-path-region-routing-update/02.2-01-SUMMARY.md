## Technical Details

### Region Routing Pattern

URL construction follows `region-routing-integration.md`:

- **Region-scoped paths** (`/bean/*`, `/reporting/*`, `/region/*`):
  - Final URL: `/api/v1/{region}{path}`
  - Example: `/bean/transactions` → `/api/v1/cn/bean/transactions`

- **Global paths** (`/auth/*`, `/symbol/*`, `/platforms/*`):
  - Final URL: `/api/v1{path}`
  - Example: `/auth/login` → `/api/v1/auth/login`

### Supported Regions

|------|------|----------|--------|

| de | 德国 | EUR | de_DE |
