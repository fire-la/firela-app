## Key Decisions

From PROJECT.md and phase summaries:

| Decision | Rationale | Outcome |
|----------|-----------|---------|
| Flutter over React Native | Single codebase for all platforms | ✓ Delivered cross-platform app |
| signals for state management | Simpler than Bloc, reactive | ✓ Effective reactive updates |
| go_router for navigation | Declarative routing, deep linking | ✓ Clean navigation structure |
| Hive for local storage | NoSQL, fast, Flutter-native | ✓ Fast local caching |
| PostHog for analytics | Open-source, self-hostable | ✓ Privacy-respecting analytics |
| patrol for UI testing | Enhanced DSL for Flutter | ✓ CI/CD integration |
| Feature-based architecture | Scalability, team autonomy | ✓ Clean code organization |
| 4-tier savings rate color coding | Finer granularity for feedback | ✓ Enhanced UX |
